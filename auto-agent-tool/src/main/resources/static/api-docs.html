AI工具治理平台API文档

AI工具治理平台的RESTful API接口

工具集管理

GET /api/tool-sets - 获取所有工具集
描述

获取所有工具集的列表

请求参数

无

响应示例
[
  {
    "id": "uuid-string",
    "name": "MCP Tools",
    "description": "Model Context Protocol Tools",
    "type": "mcp",
    "status": 1,
    "createdAt": "2023-10-01T10:00:00",
    "updatedAt": "2023-10-01T10:00:00",
    "tools": [...],
    "httpTools": [...]
  }
]
GET /api/tool-sets/{id} - 根据ID获取工具集
描述

根据ID获取特定工具集的详细信息

路径参数
  • id - 工具集ID
响应示例
{
  "id": "uuid-string",
  "name": "MCP Tools",
  "description": "Model Context Protocol Tools",
  "type": "mcp",
  "status": 1,
  "createdAt": "2023-10-01T10:00:00",
  "updatedAt": "2023-10-01T10:00:00",
  "tools": [...],
  "httpTools": [...]
}
POST /api/tool-sets - 创建工具集
描述

创建一个新的工具集

请求体
{
  "name": "新工具集名称",
  "description": "工具集描述",
  "type": "mcp",  // 或 "http"
  "status": 1     // 1-启用, 0-禁用
}
响应示例
{
  "id": "new-uuid-string",
  "name": "新工具集名称",
  "description": "工具集描述",
  "type": "mcp",
  "status": 1,
  "createdAt": "2023-10-01T10:00:00",
  "updatedAt": "2023-10-01T10:00:00",
  "tools": [],
  "httpTools": []
}
PUT /api/tool-sets/{id} - 更新工具集
描述

更新指定ID的工具集信息

路径参数
  • id - 工具集ID
请求体
{
  "name": "更新后的名称",
  "description": "更新后的描述",
  "type": "http",
  "status": 1
}
DELETE /api/tool-sets/{id} - 删除工具集
描述

删除指定ID的工具集（同时删除其下的所有工具）

路径参数
  • id - 工具集ID
PATCH /api/tool-sets/{id}/status/{status} - 更新工具集状态
描述

启用或禁用工具集

路径参数
  • id - 工具集ID
  • status - 状态值 (1-启用, 0-禁用)

工具管理

POST /api/tool-sets/{toolSetId}/tools - 添加工具到工具集
描述

向指定工具集中添加新工具

路径参数
  • toolSetId - 工具集ID
请求体
{
  "name": "工具名称",
  "description": "工具描述",
  "type": "custom",  // mcp, http, custom
  "inputSchema": "{...}",  // JSON Schema定义
  "status": 1
}

HTTP工具管理

POST /api/tool-sets/{toolSetId}/http-tools - 添加HTTP工具到工具集
描述

向指定工具集中添加新HTTP工具

路径参数
  • toolSetId - 工具集ID
请求体
{
  "name": "HTTP工具名称",
  "description": "HTTP工具描述",
  "method": "GET",  // GET, POST, PUT, DELETE
  "url": "https://api.example.com/data",
  "headers": "{...}",  // JSON格式的请求头
  "requestBodyTemplate": "{...}",  // POST/PUT请求体模板
  "responseParsingPattern": "$.data[*]",  // 响应解析模式
  "status": 1
}

实体定义

ToolSetDto
{
  "id": "string",           // 工具集唯一标识
  "name": "string",         // 工具集名称
  "description": "string",  // 工具集描述
  "type": "string",         // 工具集类型 (mcp/http)
  "status": 1,              // 状态 (1-启用, 0-禁用)
  "createdAt": "datetime",  // 创建时间
  "updatedAt": "datetime",  // 更新时间
  "tools": [...],           // 工具列表
  "httpTools": [...]        // HTTP工具列表
}
ToolDto
{
  "id": "string",           // 工具唯一标识
  "name": "string",         // 工具名称
  "description": "string",  // 工具描述
  "toolSetId": "string",    // 所属工具集ID
  "type": "string",         // 工具类型 (mcp/http/custom)
  "inputSchema": "string",  // 输入参数Schema (JSON格式)
  "status": 1,              // 状态 (1-启用, 0-禁用)
  "createdAt": "datetime",  // 创建时间
  "updatedAt": "datetime"   // 更新时间
}
HttpToolConfigDto
{
  "id": "string",                    // HTTP工具唯一标识
  "name": "string",                  // HTTP工具名称
  "description": "string",           // HTTP工具描述
  "toolSetId": "string",             // 所属工具集ID
  "method": "string",                // HTTP方法 (GET/POST/PUT/DELETE)
  "url": "string",                   // API URL模板
  "headers": "string",               // 请求头 (JSON格式)
  "requestBodyTemplate": "string",   // 请求体模板 (JSON格式)
  "responseParsingPattern": "string", // 响应解析模式
  "status": 1,                       // 状态 (1-启用, 0-禁用)
  "createdAt": "datetime",           // 创建时间
  "updatedAt": "datetime"            // 更新时间
}

使用示例

创建MCP工具集
# 创建请求
curl -X POST http://localhost:17000/api/tool-sets \
  -H "Content-Type: application/json" \
  -d '{
    "name": "我的MCP工具集",
    "description": "包含多个MCP协议工具",
    "type": "mcp",
    "status": 1
  }'

# 添加工具到工具集
curl -X POST http://localhost:17000/api/tool-sets/{toolSetId}/tools \
  -H "Content-Type: application/json" \
  -d '{
    "name": "数据查询工具",
    "description": "用于查询业务数据",
    "type": "mcp",
    "inputSchema": "{...}",
    "status": 1
  }'
